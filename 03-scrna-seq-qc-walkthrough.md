## Part 2: Load the Data

Read the 10x data and create a Seurat object:

```r
# Load the filtered feature-barcode matrix
pbmc.data <- Read10X_h5("data/filtered_feature_bc_matrix.h5")

# Create a Seurat object
pbmc <- CreateSeuratObject(
    counts = pbmc.data,
    project = "PBMC",
    min.cells = 3,
    min.features = 200
)

# View a summary
pbmc
```

**What the parameters mean:**
- `min.cells = 3`: Only keep genes detected in at least 3 cells
- `min.features = 200`: Only keep cells with at least 200 detected genes

> **Key concept:** Your Seurat object is a matrix where:
> - **Rows** = genes
> - **Columns** = cells  
> - **Values** = UMI counts (number of transcripts detected)

## Part 3: Calculate QC Metrics

Single-cell data contains technical artifacts we need to identify and filter out. We'll calculate metrics to detect:
- Empty droplets (too few genes)
- Doublets (too many genes)
- Dying cells (high mitochondrial content)

### Add Mitochondrial Percentage

Dying or stressed cells leak cytoplasmic RNA but retain mitochondrial RNA, leading to high mitochondrial gene percentages:

```r
pbmc[["percent.mt"]] <- PercentageFeatureSet(pbmc, pattern = "^MT-")
```

### (Optional) Add Ribosomal Percentage

```r
pbmc[["percent.ribo"]] <- PercentageFeatureSet(pbmc, pattern = "^RPL|^RPS")
```

---

## Part 4: Visualize QC Metrics

Before filtering, visualize the distribution of QC metrics across all cells:

### Violin Plots

```r
VlnPlot(
    pbmc,
    features = c("nFeature_RNA", "nCount_RNA", "percent.mt"),
    ncol = 3
)
```

**What to look for:**
- `nFeature_RNA`: Number of genes detected per cell
- `nCount_RNA`: Total UMI counts per cell
- `percent.mt`: Percentage of reads from mitochondrial genes

### Scatter Plots

These help identify relationships between metrics:

```r
# Genes vs. counts (look for doublets in upper right)
FeatureScatter(pbmc, feature1 = "nCount_RNA", feature2 = "nFeature_RNA")

# Counts vs. mitochondrial percentage (dying cells have high mito)
FeatureScatter(pbmc, feature1 = "nCount_RNA", feature2 = "percent.mt")
```

**Interpreting the plots:**

| Pattern | Likely Interpretation |
|---------|----------------------|
| Low genes + low counts | Empty droplets or debris |
| Very high genes + very high counts | Possible doublets (two cells in one droplet) |
| High mitochondrial % | Stressed or dying cells |

---

## Part 5: Filter Low-Quality Cells

Based on the QC visualizations, apply filtering thresholds. These values work well for PBMCs:

```r
pbmc <- subset(
    pbmc,
    subset = nFeature_RNA > 200 &
             nFeature_RNA < 3000 &
             percent.mt < 10
)
```

**Understanding the thresholds:**
- `nFeature_RNA > 200`: Remove empty droplets
- `nFeature_RNA < 3000`: Remove potential doublets
- `percent.mt < 10`: Remove dying/stressed cells

> **Important:** These thresholds are not universal! Different tissues and experimental conditions may require different cutoffs. PBMCs generally tolerate slightly higher mitochondrial content than solid tissues.

## Part 7: Normalize the Data

Normalization adjusts for differences in sequencing depth between cells, making them comparable:

```r
pbmc <- NormalizeData(
    pbmc,
    normalization.method = "LogNormalize",
    scale.factor = 10000
)
```

**What this does:**
1. Divides each cell's counts by its total counts
2. Multiplies by a scale factor (10,000)
3. Log-transforms the result

> **Key concept:** Normalization makes cells comparable but does not remove biological variation—that's what we want to study!

---

## Part 8: Identify Highly Variable Genes

Not all genes are equally informative. We identify genes that vary substantially across cells—these drive the differences between cell types:

```r
pbmc <- FindVariableFeatures(
    pbmc,
    selection.method = "vst",
    nfeatures = 2000
)
```

### Visualize Variable Features

```r
# Plot variance vs. mean expression
VariableFeaturePlot(pbmc)
```

### Label the Top Variable Genes

```r
# Get top 10 most variable genes
top10 <- head(VariableFeatures(pbmc), 10)

# Add labels to the plot
LabelPoints(
    plot = VariableFeaturePlot(pbmc),
    points = top10,
    repel = TRUE
)
```

These highly variable genes will be used for downstream dimensionality reduction and clustering.
